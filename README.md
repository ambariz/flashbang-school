# Flashbang School

A military style puzzle visual novel built with the Ren'Py engine.  
You are a trainee inside a restricted facility where every decision is a test of logic, memory, and reaction.

[![Click to Play](https://img.shields.io/badge/CLICK%20ME%20TO%20PLAY-itch.io-red?style=for-the-badge&logo=itchdotio)](https://ambariz.itch.io/flashbang-school)

**LINK :** https://ambariz.itch.io/flashbang-school  

---

## Gameplay

- Enter a military training facility as a new recruit  
- Solve security code puzzles as encrypted locks  
- Remember your assigned ID and use it later  
- Make fast decisions in scenarios  
- Wrong answers trigger a flashbang reset  

Each run feels slightly different as you learn from your mistakes.

---
## Features

- Multiple missions with increasing difficulty  
- Simple menu-based controls  
- Replay system with persistent memory  
- Hidden secret ending  
- Military UI
